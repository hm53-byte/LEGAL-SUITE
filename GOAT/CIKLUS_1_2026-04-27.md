## Korak 1 — TRŽIŠNI ISTRAŽIVAČ (subagent, prazan context, s WebSearch)

Subagent s `general-purpose` role-om i pristupom `WebSearch` + `WebFetch` istražio 3 platforme (Paddle, LemonSqueezy, Polar.sh) po 4 kriterija:

- K1 Provizije
- K2 API docs + Cloudflare Workers kompatibilnost
- K3 HR IBAN d.o.o. payout
- K4 Subscription model

**Output**: 30+ verifikabilnih URL referenci iz vendor docs + treća strana (Dodo Payments, swell.is, fungies.io, TechCrunch, UserJot).

**Ključni nalazi**:

### Paddle
- 5% + $0.50 baseline, jedan globalni rate
- HR podržan kao seller country
- $100 min payout, mjesečno
- "No deprecation" API politika
- Min ticket <$10 traži custom pricing

### LemonSqueezy
- 5% + 1.5% (non-US card) + 0.5% (subscription) = **7% + $0.50** efektivno za EU sub
- **Stripe akvizicija srpanj 2024** + roadmap Stripe Managed Payments → tim u siječnju 2026 priznao "slower support and fewer product updates"
- HR IBAN nije eksplicit potvrđen
- Web Crypto API kompatibilan (dobar CF Workers fit)

### Polar.sh
- 4% + 1.5% (non-US) + 0.5% (sub) = **6% + $0.40** efektivno za EU sub
- Croatia eksplicit listana kao supported seller country
- Stripe Connect Express u pozadini (konfigurabilan min payout, on-demand)
- Open-source, aktivni development
- API versioning policy nije publicly dokumentirana (rizik)
- Manja public usage history

## Korak 3 — REVIZOR (subagent, prazan context, neovisan fact-check)

Subagent s 4 nasumično odabranih tvrdnji za fact-check:
- T1 Paddle effective fee = 5% + $0.50 (jedan globalni rate)
- T2 LMSQZY Stripe akvizicija + 2026 priznanje
- T3 Polar 6% + $0.40 aditivno
- T4 Polar HR IBAN + €13 min payout

**Rezultat**: 2/4 PASS, 2/4 PARTIAL/DIVERGENCE.

### Kritične korekcije

1. **T1 DIVERGENCE — Paddle FX spread**: Dodo Payments (treća strana) potvrđuje **2-3% FX margin spread** na non-USD payout. Headline 5% + $0.50 nije puna slika za HR EUR seller koji prima EUR payouts kroz Paddle USD account.

2. **T3 PASS s napomenom**: Polar 6% + $0.40 točno za EU sub, ali matrica treba dodati Stripe payout pass-through fees ($2/mj fixed + 0.25% + $0.25/payout + cross-border 0.25-1%).

3. **T4 PARTIAL**: Croatia supported preko Stripe Connect Express (PASS), ali €13 min payout je krivo prevedena vrijednost iz $13 USD. Stvarno je **konfigurabilan** preko Stripe Connect (default €1 base unit).

4. **T2 PASS s warning**: LMSQZY akvizicija PASS, ali doslovan citat 2026 priznanja samo via sekundarni izvor (Dodo Payments parafraza). Akvizicijski narativ stoji.

## Korak 4 — Refrakcija (orkestrator)

VAGAR matrica revidirana s REVIZOR korekcijama. **Glavna promjena**: K1 i K3 ocjene rebalansirane jer Paddle FX spread (2-3%) eliminira njegovu fee prednost za HR EUR sellera.

| K2 CF Workers + API | 20% | **5** ("no deprecation") | 4 | 3 (versioning nedokumentiran) |
| K3 HR IBAN d.o.o. | 25% | 3 ($100 min, FX pri payout) | 2 | **5** (Stripe Connect, konfigurabilan, on-demand) |
| K4 Long-term viability | 15% | **5** | **1** | 3 |

**Score V2**:
- **Polar.sh: 4.20 / 5** ← pobjednik
- Paddle: 3.90 / 5
- LemonSqueezy: 2.45 / 5

### Revidirani ROI (HR EUR seller, 100 PRO useri × 9€/mj × 12 mj = 10 800€/god prihod, 1200 tx, 12 payouts)

| Platforma | Base fee | Per-tx | FX/Stripe pass-through | **Total/god** | **Neto/god** |
|---|---|---|---|---|---|
| **Polar.sh** | 648€ | ~480€ | ~80€ Stripe pass-through | **~1 208€** | **~9 592€** |
| Paddle | 540€ | ~600€ | **216-324€ FX spread** | ~1 356-1 464€ | ~9 336-9 444€ |
| LemonSqueezy | 756€ | ~600€ | nepoznat FX | ~1 356€+ | <9 444€ |

→ Polar **~150-250€/god jeftiniji** od Paddle-a za HR EUR sellera.
